# Vue-thinkjs
商城后台管理系统
前端：vue-admin-template；后端：thinkjs；数据库：mysql

除了包含基本的增删查改，还有用户权限的验证，图片上传，连表查询

### mallserver:后端   
npm install
npm run dev

### vue-admin-template:前端  
npm install
npm start

### mall.sql数据库文档
原型为cmswing.sql，没做改动
